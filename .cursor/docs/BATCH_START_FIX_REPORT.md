### 1.2 Устранена ошибка 404 при «Запустить» / «Завершить»

**Причина:** В части окружений (uvicorn, Docker) маршрут `PATCH /api/v1/batches/{batch_id}` мог сопоставляться раньше, чем более специфичные пути с суффиксами `/start` и `/complete`, из‑за чего запросы к этим действиям не находили обработчик и возвращали 404.

**Решение:**

| Компонент | Изменения |
|-----------|-----------|
| **Backend** | Маршруты действий вынесены в отдельный роутер `router_actions` с путями **`/start/{batch_id}`** и **`/complete/{batch_id}`** (литерал в пути до UUID). В `main.py` этот роутер подключается **первым**, затем основной роутер батчей — порядок сопоставления маршрутов гарантирован. |
| **Frontend** | В `api.ts`: вызовы переведены на `/batches/start/${id}` и `/batches/complete/${id}`; добавлена нормализация базового URL API (добавление суффикса `/api/v1` при необходимости). |
| **Тесты** | В `test_batches_api.py` добавлены тесты `test_start_batch_endpoint` и `test_complete_batch_endpoint` для эндпоинтов `PATCH /api/v1/batches/start/{id}` и `PATCH /api/v1/batches/complete/{id}`. |

**Итоговые эндпоинты:**

- Запуск батча: `PATCH /api/v1/batches/start/{batch_id}`
- Завершение батча: `PATCH /api/v1/batches/complete/{batch_id}`

## 3. Проверка и окружение

- **Автотесты:** тесты API батчей (в т.ч. `test_start_batch_endpoint`, `test_complete_batch_endpoint`) проходят (9 passed в `test_batches_api.py`).
- **Ручная проверка:** после пересборки backend в Docker и перезагрузки фронта кнопки «Запустить» и «Завершить» отрабатывают корректно; в Network видны ответы 200 и обновление статусов партий (PLANNED → IN_PROGRESS → COMPLETED).

**Важно для деплоя:** Backend в Docker использует образ, в который при сборке копируется `backend/core`. Для применения правок маршрутов после изменений в коде необходимо пересборка и перезапуск:

```bash
docker-compose up -d --build
```

или по шагам: `docker-compose build backend` и затем `docker-compose up -d`. Без `--build` / отдельного `build backend` контейнер продолжит работать со старым кодом из образа.
